# 谷歌生草机介绍：
***
## 简介：
最近谷歌翻译在网上火了，因为谷歌翻译是机器翻译，所以如果只是翻译一次，大体还是有九成九的准确度的，但是当使用谷歌翻译选择小语种，然后小语种再翻译到另一个小语种，重复翻译多遍后，内容就会疯狂失真，然后准确度也会随着翻译次数开始偏离原来的真正的意思。  
__可如果手动翻译效率极低，于是就有了这款自动翻译机__  
这是用python做的谷歌自动翻译机，可以自动翻译内容*十几甚至几十遍~~就是翻译太多变会卡~~*
